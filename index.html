今まで　vs　このツール

今まで

手入力で経費登録

  • レシートを見ながら1件ずつ入力
  • 日付・金額を打ち間違える
  • まとめてやると時間がかかる
  • 入力漏れが発生しやすい
このツール

写真を撮るだけ

  • スマホで撮影するだけでOK
  • AIが自動で項目を読み取る
  • 複数枚をまとめて一括処理
  • 弥生にそのまま取り込める

ツールの起動方法

以下のコマンドをターミナルに貼り付けて実行してください。ブラウザが自動で開きます。

# 初回のみ（依存パッケージのインストール）
pip install -r requirements.txt

# ツールを起動する
streamlit run app.py
